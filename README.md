# Viom Air Cargo

Specialized handling for exquisite and delicate items.

## Project info

Viom Air Cargo is a premier air freight service specializing in the transportation of high-value, delicate goods such as luxury jewelry, fine glassware, and sensitive electronics.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd viom-air-cargo-landing
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```
